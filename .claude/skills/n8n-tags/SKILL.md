Interact with n8n tags via CLI. Run commands with `uv run n8n_cli.py tags <action>`.

## Commands

```
tags list
tags get <id>
tags create <name>
tags update <id> <name>
tags delete <id>
```

Present results in a clean readable format. $ARGUMENTS
